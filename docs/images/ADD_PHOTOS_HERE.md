# Project Photos

Add your photos here with these exact filenames so the README loads them automatically:

| Filename | What to put there |
|----------|-------------------|
| `cover.jpg` | Best overall shot of the wheelchair |
| `welding.jpg` | Photo from the welder |
| `fabrication.jpg` | The wheel mount/elevation fix |
| `wiring.jpg` | Electronics and wiring work |
| `testing.jpg` | Component testing / troubleshooting |
| `presentation.jpg` | Your RAIN presentation |
| `digital-twin.png` | Screenshot of the digital twin running |

Just drop the photos into this folder with those exact names and they will appear in the README automatically.
